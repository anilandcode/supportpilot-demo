# Integrations

## GitHub

The GitHub integration links pull requests to issues, moves issues to In Review when a pull request opens, and closes issues when a pull request is merged. Users can create issues from GitHub comments. The integration requires installing the AcmeDesk GitHub App on the organization or repository.

## Slack

The Slack integration lets teams create, update, and receive notifications about AcmeDesk issues from Slack. Users can run /acmedesk to create an issue, receive issue notifications in a channel, unfurl AcmeDesk links, and configure notification rules by team or project.

## Figma

The Figma integration embeds frames, prototypes, and file previews in issue descriptions and comments. It activates automatically when a Figma URL is pasted. The Figma file must allow anyone with the link to view it.

## Notion

The Notion integration creates a two-way sync between AcmeDesk issues and Notion database items. Status updates sync between both tools. The integration is configured per workspace and requires a Notion workspace admin to authorize the connection. Notion sync is available on Pro and Business plans.

## Other integrations

AcmeDesk also integrates with Sentry, Zendesk, Intercom, PagerDuty, Zapier, REST APIs, and webhooks.
