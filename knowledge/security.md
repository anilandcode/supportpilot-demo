# Security and Privacy

## SOC 2

AcmeDesk is SOC 2 Type II certified. The audit covers Security, Availability, and Confidentiality. The latest audit report is available to Business and Enterprise customers under NDA by opening a support ticket with the subject SOC 2 Report Request.

## Encryption

All customer data is encrypted at rest with AES-256 and in transit with TLS 1.2 or higher. Database backups are encrypted. Passwords are hashed with bcrypt using a minimum cost factor of 12. Encryption keys are managed through AWS KMS with automatic rotation.

## Single Sign-On

SAML 2.0 SSO is available on Business and Enterprise plans. AcmeDesk supports Okta, Azure Active Directory, Google Workspace, and OneLogin. Admins can enforce SSO for all members or allow a grace period during migration.

## Role-Based Access Control

Business and Enterprise workspaces support custom roles beyond Member, Admin, and Guest. Admins can create roles with granular permissions for issue creation, project management, billing access, API key creation, and integration management.

## GDPR

AcmeDesk is GDPR compliant. Data is stored in the United States by default. Enterprise customers can choose EU data residency in Frankfurt, Germany. AcmeDesk provides a Data Processing Agreement on request and completes data deletion requests within 30 days.
